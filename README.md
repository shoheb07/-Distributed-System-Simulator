# -Distributed-System-Simulator
The Distributed System Simulator is an educational project developed using Python to simulate distributed computing concepts and communication models.

# Distributed System Simulator

## Project Overview

The Distributed System Simulator demonstrates distributed computing concepts such as election algorithms, client-server communication, and process coordination.

The project is developed using Python socket programming.

# Features

- Bully election algorithm
- Ring election algorithm
- Client-server communication
- Coordinator election
- Distributed process simulation

# Technologies Used

- Python
- Socket Programming

# Run Project

## Run Server

python server.py

## Run Client

python client.py
